Ce este algoritmul Fill?

Acest algoritm este un algoritm ce se folosește de obicei atunci când avem de lucru cu o matrice. Ce face acest algoritm? Are multe aplicații, dar una din cele mai practice este atunci când determinăm numărul elementelor dintr-o matrice care respectă o anumită condiție.

Prezentare generală

Algoritmul de fill este un algoritm special prin care parcurgem matricea și determinăm câte părți distincte avem într-o problemă.
